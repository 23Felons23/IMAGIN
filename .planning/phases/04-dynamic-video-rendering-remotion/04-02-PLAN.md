# Plan 04-02: Bouncy Subtitle & Emoji Overlay Components

## Objective
Implement the word-by-word spring-animated subtitle system and the rule-based emoji keyword overlay component, both fully synced to Remotion's frame timeline.

## Tasks

<task>
<read_first>
- remotion/src/PodcastClip.tsx
- .planning/phases/04-dynamic-video-rendering-remotion/04-CONTEXT.md
</read_first>
<action>
Create `remotion/src/Subtitles.tsx`.

Props: `words: {word: string, start: number, end: number, speaker: string}[]`

Implementation:
- Use `useCurrentFrame()` and `useVideoConfig()` to get current time in seconds: `const currentTime = frame / fps`
- Find the active word: `words.find(w => currentTime >= w.start && currentTime <= w.end)`
- Find the previous 3–5 words for context display (shown greyed out)
- For the active word: apply a `spring()` animation for scale from 0.6 → 1.0, starting at the word's `start` time converted to frames
- Style:
  ```tsx
  // Active word
  color: '#FF3366'         // accent
  fontSize: 72
  fontWeight: 900
  textShadow: '-3px -3px 0 #000, 3px -3px 0 #000, -3px 3px 0 #000, 3px 3px 0 #000'  // thick stroke
  transform: `scale(${springValue})`
  
  // Context words (previous)
  color: 'white'
  opacity: 0.5
  fontSize: 60
  ```
- Position the subtitle block in the lower third (bottom 35% of frame), centered horizontally
- Each line max ~4–5 words, wrap onto new line automatically
</action>
<acceptance_criteria>
- Active word scales from ~0.6→1.0 using Remotion's `spring()`
- Previous words visible but greyed out for context
- Text has thick black stroke simulating the Hormozi style
</acceptance_criteria>
</task>

<action>
Create `remotion/src/emojis.ts` — the keyword dictionary:
```ts
export const EMOJI_MAP: Record<string, string> = {
  money: '💰', rich: '💰', wealth: '💰', profit: '💸',
  fire: '🔥', hot: '🔥', insane: '🤯', crazy: '🤯', wild: '🤯',
  love: '❤️', heart: '❤️',
  win: '🏆', winner: '🏆', champion: '🏆',
  rocket: '🚀', launch: '🚀', growth: '📈',
  god: '🙏', amazing: '🙌', wow: '😱',
  hundred: '💯', percent: '💯',
  brain: '🧠', smart: '🧠', genius: '🧠',
  fast: '⚡', speed: '⚡',
  stop: '🛑', wait: '✋',
};

export function getEmojiForWord(word: string): string | null {
  return EMOJI_MAP[word.toLowerCase().replace(/[^a-z]/g, '')] ?? null;
}
```

Create `remotion/src/EmojiOverlay.tsx`:
- Props: `words`, same as Subtitles
- For the active word, call `getEmojiForWord(word)` — if match, render the emoji
- Animate: emoji enters with `spring()` scale 0→1.2→1.0 over 8 frames, then fades out after 12 frames using `interpolate()`
- Position: centered above the subtitle block
- Font size: 96px

Add `<Subtitles>` and `<EmojiOverlay>` to `PodcastClip.tsx` as overlaid children with `position: absolute`.
</action>
<acceptance_criteria>
- `emojis.ts` exports `EMOJI_MAP` and `getEmojiForWord()`
- Emoji appears on matching word and animates with spring scale
- Emoji fades out after ~12 frames using `interpolate()`
</acceptance_criteria>
</task>
